Typing Spanish Characters

Windows

To type in Spanish, you could set up an international keyboard on your computer. However, if you don't wish to do that, you can instead use ASCII codes. To do so, hold down the ALT key and then type the appropriate code. The following lists shows the combinations for the different characters.

á Alt + 160 Á Alt + 0193
é Alt + 130 É Alt + 0201
í Alt + 161 Í Alt + 0205
ó Alt + 162 Ó Alt + 0211
ú Alt + 163 Ú Alt + 0218
ñ Alt + 164 Ñ Alt + 165
ü Alt + 0252 Ü Alt + 0220
¡ Alt + 173 ¿ Alt + 168

Note: laptop users might need to take additional steps such as pressing the Fn key. If the above does not work, you can try searching for additional info specific to your computer.

Mac

Using Spanish characters on a Mac is simpler than on Windows. For accented vowels, hold down the option and e keys, release, and then press the key corresponding to the vowel you want to accent. For ñ, hold down option and n, release, and then hit n again. The following chart shows the specific combinations.

á Option + e, then A Á Option + e, then Shift + A
é Option + e, then E É Option + e, then Shift + E
í Option + e, then I Í Option + e, then Shift + I
ó Option + e, then O Ó Option + e, then Shift + O
ú Option + e, then U Ú Option + e, then Shift + U
ñ Option + n, then N Ñ Option + n, then Shift + N
ü Option + u, then U Ü Option + u, then Shift + U
¡ Option + 1 ¿ Option + ?
